# 命名规范 (Google C++ Style)

- 变量名：`snake_case`（如 `frame_count`）
- 函数名：`CamelCase`（如 `GetFrameCount`）
- 类名：`PascalCase`（如 `RobotKinematics`）
- 私有成员变量：`snake_case` + 下划线后缀（如 `member_variable_`）
- 常量：`kCamelCase`（如 `kMaxIterations`）
- 枚举值：`kCamelCase`（如 `kDeg`, `kRad`）
- 命名空间：全小写（如 `nl::kinematics`）
- 宏：`ALL_CAPS`（如 `GRINDING_UI_EXPORT`）
